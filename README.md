# modcasts

phpbb modding screencasts

modcasts is a site with screencasts on phpBB modding.

# powered by

* doctrine 2
* symfony 2 (some components)
* twig

# install

1. adjust configuration in services.yml
2. initialize the db with ./doctrine orm:schem-tool:create
3. PROFIT!
